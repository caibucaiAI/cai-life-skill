# cai-life-skill · 人生系统 ecosystem

一套基于三层文件结构的自我认知 skill——把日常事件 / 心理规律 / 人生叙事分开存放又自动联动，让 AI 真正了解你这个人，而不是每次都从零认识。

适用于 Claude Code / Claude.ai / Codex。

---

## 这是什么

很多人用 AI 做心理疏导、做决策辅助、做内容创作时都遇到过同一个问题：每次都要从头介绍"我是谁"，AI 只能拿到碎片，给的建议也漂在表面。

这套 skill 的解法是——把你这个人按三层结构存进本地文件，让 AI 每次都能调用完整的你：

| 层 | 文件 | 装什么 |
|---|------|--------|
| **事件层** | `经历流水账.md` | 发生在你身上的事，倒序，随手记 |
| **规律层** | `模式与盲点.md` | 你反复出现的行为模式 / 容易栽的坑 / 情绪触发 |
| **叙事层** | `我的故事.md` | 你这个人是谁，你的判断框架、价值观、口头禅 |

三层不重复——流水账是收件箱，模式与盲点是雷达，我的故事是图书馆。事件流入，规律提炼，叙事沉淀。

## ecosystem 完整结构（v2）

```
cai-life-skill/
├── SKILL.md                    ← 主 skill：life-os（人生系统核心）
├── README.md                   ← 本文件
└── sub-skills/
    ├── cai-money/
    │   └── SKILL.md            ← life-money：财务模块
    ├── cai-health/
    │   └── SKILL.md            ← life-health：健康模块
    ├── cai-jobhunt/
    │   └── SKILL.md            ← life-jobhunt：求职模块
    └── love-coach/
        └── SKILL.md            ← life-love：爱情/亲密关系模块
```

### 5 个模块说明

| 模块 | skill name | 管什么 |
|------|-----------|--------|
| **life-os**（主） | `life-os` / `cai-life-skill` | 核心系统：自我认知、决策框架、心理盲点、周复盘 |
| **life-money** | `life-money` | 财务的事：财务知识、报价决策、接单判断 |
| **life-health** | `life-health` | 健康的事：体检报告、症状日志、就诊归档、健康决策 |
| **life-jobhunt** | `life-jobhunt` | 求职的事：公司调研、面试纪要、offer 决策、行业 insight |
| **life-love** | `life-love` | 爱情的事：亲密关系分析、感情决策、盲区联动 |

**设计原则**：每个子 skill 管自己领域的"事"，life-os 管所有领域背后的"我"——盲区识别、判断框架、书摘提醒，都在主系统里。子 skill 的决策最终都写回 life-os 的 `经历流水账.md`。

---

## 核心机制

- **3 个标签**：`[决策]` / `[模式]` / `[洞察]`，记流水账时用
- **❌/✅ 符号**：盲区再现 vs 突破，进步轨迹一眼可见
- **书摘联动**（可选）：把读过的书的划线，按 ❌/✅ 配对到对应盲区——下次又栽进同一个坑，递给你的不是 AI 鸡汤，是你自己曾经划过的那句话
- **统一原则**：先写后看，事后可删，不阻塞节奏

## 安装

### 主 skill（必装）

```bash
cd ~/.claude/skills
git clone https://github.com/caibucaiAI/cai-life-skill.git
```

重启 Claude Code / Codex 会话，输入：
> 「开始使用人生系统 skill」

skill 会自动接管，5 分钟带你搭好自己的版本。

### 子 skill（按需装）

把对应子目录的 `SKILL.md` 复制到你的 `~/.claude/skills/` 下对应子文件夹，或直接引用 repo 里的路径。

**示例：单独装财务模块**
```bash
mkdir -p ~/.claude/skills/life-money
cp ~/.claude/skills/cai-life-skill/sub-skills/cai-money/SKILL.md ~/.claude/skills/life-money/SKILL.md
```

**示例：单独装健康模块**
```bash
mkdir -p ~/.claude/skills/life-health
cp ~/.claude/skills/cai-life-skill/sub-skills/cai-health/SKILL.md ~/.claude/skills/life-health/SKILL.md
```

子 skill 的触发词：

| 模块 | 触发词示例 |
|------|----------|
| life-money | 「财务」「报价」「这个值不值得接」「记进财务skill」 |
| life-health | 「整理体检报告」「记一下症状」「该不该停下来休息」 |
| life-jobhunt | 「我下周面试XX」「整理面试笔记」「这个offer要不要接」 |
| life-love | 「恋爱」「亲密关系」「感情决策」「我有点喜欢XX」 |

## 5 个问题预览（首次使用主 skill 的流程）

skill 会按顺序问你 5 个问题，每一步都会解释为什么这么问：

📍 **Q1：你的昵称是？**
→ AI 后面会一直叫你这个名字

📍 **Q2：你想把文件放在哪里？**
- A. Obsidian（推荐）/ B. 桌面试试 / C. 自定路径

📍 **Q3：⭐ 最关键的一步——有没有关于"你这个人"的资料可以投喂？**
- 简历 / 自我介绍 / 自我总结 / MBTI / 测评 / 朋友给你的画像 / 成长经历……
- AI 会从中读出你的初版「模式与盲点」

📍 **Q4：装了 weread-skills 吗？**（微信读书划线联动）

📍 **Q5：试一句「记一下流水账，今天 ___」**

## 装好之后日常怎么用

说话就行，不用记命令。常用场景：

- 🔍 **周复盘**：「周复盘」→ 时间饼图 + 多维方向照镜子
- 🎯 **辅助决策**：「帮我做决策」→ 用你自己的判断框架给结论
- 📚 **沉淀叙事**：「merge 一下」→ 把想清楚的事融进我的故事
- 💰 **财务**：「这个值不值得接」→ life-money 财务决策
- 🏥 **健康**：「我体检完了」→ life-health 整理报告
- 💼 **求职**：「我下周面试XX」→ life-jobhunt 调研报告
- ❤️ **感情**：「我有点喜欢XX」→ life-love 联动盲区分析

## 隐私

**所有文件在你本地。** AI 处理时会把内容作为上下文发送给模型，但文件本身始终在你的硬盘上。

建议：
- 敏感人物用代号（"M""J""那个朋友"）
- 真正不想让任何系统看到的内容，**不记进系统**
- 这是给你"看见模式"的工具，不是"逐字真相"的存档
