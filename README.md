# cai-life-skill · 人生系统

一套基于三层文件结构的自我认知 skill——把日常事件 / 心理规律 / 人生叙事分开存放又自动联动，让 AI 真正了解你这个人，而不是每次都从零认识。

适用于 Claude Code / Claude.ai / Codex。

---

## 这是什么

很多人用 AI 做心理疏导、做决策辅助、做内容创作时都遇到过同一个问题：每次都要从头介绍"我是谁"，AI 只能拿到碎片，给的建议也漂在表面。

这套 skill 的解法是——把你这个人按三层结构存进本地文件，让 AI 每次都能调用完整的你：

| 层 | 文件 | 装什么 |
|---|------|--------|
| **事件层** | `经历流水账.md` | 发生在你身上的事，倒序，随手记 |
| **规律层** | `模式与盲点.md` | 你反复出现的行为模式 / 容易栽的坑 / 情绪触发 |
| **叙事层** | `我的故事.md` | 你这个人是谁，你的判断框架、价值观、口头禅 |

三层不重复——流水账是收件箱，模式与盲点是雷达，我的故事是图书馆。事件流入，规律提炼，叙事沉淀。

---

## 核心机制

- **3 个标签**：`[决策]` / `[模式]` / `[洞察]`，记流水账时用
- **❌/✅ 符号**：盲区再现 vs 突破，进步轨迹一眼可见
- **书摘联动**（可选）：把读过的书的划线，按 ❌/✅ 配对到对应盲区——下次又栽进同一个坑，递给你的不是 AI 鸡汤，是你自己曾经划过的那句话
- **统一原则**：先写后看，事后可删，不阻塞节奏

## 安装

### 方式 1：git clone（推荐）

```bash
cd ~/.claude/skills
git clone https://github.com/caibucaiAI/cai-life-skill.git
```

### 方式 2：手动下载

1. 从这个仓库下载 ZIP（绿色 `Code` 按钮 → Download ZIP）
2. 解压
3. 把整个 `cai-life-skill` 文件夹放进 `~/.claude/skills/`

> Mac 找不到这个文件夹的话：打开 Finder，按 `cmd + shift + g`，输入 `~/.claude/skills` 直达。

### 启动

重启 Claude Code / Codex 会话，输入：

> 「开始使用人生系统 skill」

skill 会自动接管，5 分钟带你搭好自己的版本。

## 5 个问题预览（首次使用流程）

skill 会按顺序问你 5 个问题，每一步都会解释为什么这么问：

📍 **Q1：你的昵称是？**
→ AI 后面会一直叫你这个名字

📍 **Q2：你想把文件放在哪里？**
- A. Obsidian（推荐）/ B. 桌面试试 / C. 自定路径
- 没装 Obsidian 可以先下：https://obsidian.md

📍 **Q3：⭐ 最关键的一步——有没有关于"你这个人"的资料可以投喂？**
- 简历 / 自我介绍 / 自我总结 / MBTI / 测评 / 朋友给你的画像 / 成长经历……
- AI 会从中读出你的初版「模式与盲点」

> 💡 重要提示：投喂得越全，AI 读你越准。
> 只发简历的话，AI 只能看到"做成功的那一面"，画像会偏。最好同时有：
> ✅ 你做成过什么、闪光的事、擅长的部分
> ❌ 你栽过什么坑、低谷、情绪反复出现的地方
> **正向 + 负向都有，画像才完整。**

📍 **Q4：装了 weread-skills 吗？**
- A. 装了 → 拉取读书划线，按 ❌/✅ 配对到对应盲区
- B. 没装但想装 → AI 给你安装步骤
- C. 跳过

📍 **Q5：试一句「记一下流水账，今天 ___」**
→ AI 自动识别这件事是「决策」「模式」还是「洞察」，对应联动模式与盲点

## 装好之后日常怎么用

搭好之后日常使用就是说话——不用记任何命令、不用关心文件怎么变。

skill 还会告诉你这些场景可以探索：

- 🔍 **周复盘**：说「周复盘」，AI 用「递镜模式」帮你照镜子（时间饼图 + 多维方向）
- 🎯 **辅助决策**：说「帮我做决策」，AI 用你自己写过的判断框架给结论（不是给一堆选项让你纠结）
- 📚 **沉淀叙事**：当你想清楚一件事，AI 会主动问要不要 merge 进「我的故事」——慢慢，你这个人就会从碎片变成一本书

---

## 隐私

**所有文件在你本地。** AI 处理时会把内容作为上下文发送给模型，但文件本身始终在你的硬盘上。

建议：
- 敏感人物用代号（"M""J""那个朋友"）
- 真正不想让任何系统看到的内容，**不记进系统**
- 这是给你"看见模式"的工具，不是"逐字真相"的存档
